# Aurevyn — Premium South African E-Commerce Frontend

> A handcrafted premium multi-page e-commerce frontend built with vanilla HTML5, CSS3, and JavaScript — no frameworks or build tools.

---

## 🌐 Live Demo

🔗 https://aurevyn.netlify.app/

---

## ✨ Project Overview

Aurevyn is a fictional premium South African online shopping platform showcasing fashion, sneakers, electronics, smart devices, accessories, and lifestyle products.

The project was designed as a modern frontend portfolio project focused on elegant UI/UX, responsiveness, animations, and smooth shopping interactions using only core web technologies.

---

## 🚀 Features

- Premium modern UI design
- Fully responsive layout
- Sticky glassmorphism navigation
- Mobile hamburger menu
- Hero section with featured products
- Product category grid
- Featured & trending products
- Product quick-view modal
- Shopping cart drawer
- Wishlist functionality
- Product search and filtering
- Checkout page with validation
- Contact form validation
- FAQ accordion
- Testimonial slider
- Newsletter subscription
- Scroll animations & micro-interactions
- LocalStorage persistence
- South African Rand (R) pricing

---

## 🧰 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Netlify Deployment
- Visual Studio Code
- Responsive Web Design

- Unsplash Images

## 💻 Run Locally

### Option 1 — Open Directly

```bash

```

### Option 2 — VS Code Live Server

1. Open the project in Visual Studio Code
2. Install the Live Server extension
3. Right-click `index.html`
4. Select **Open with Live Server**
